# Pharm Bot

Telegram-бот для работы с Google Sheets (pharm проект).

## Возможности

- Получение данных из Google Таблицы
- Удобное взаимодействие через Telegram
- Автоматический деплой через GitHub Actions + Docker

## Технологии

- Python 3.12
- aiogram 3
- Google API (Sheets)
- Docker
- GitHub Actions (CI/CD)
